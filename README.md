# zFile
### Made by [Poienariu Vlad](https://www.linkedin.com/in/poienariu-vlad/)

This is a open-source and free to use project made by Poienariu Vlad, this app comes in handy when you are in need of renaming many files by a criteria or not, for example having a lot of images with random names, by using the Increment function you can name them Photo 1-2-3-4 an so on in just few clicks.
zFiles is built using Java programming language together with Swift UI and Flatpat for design.

## 🚀 Features
- Allow to rename multiple file easily
- LowerCase
- UpperCase
- Remove phase
- Add phase
- Increment

## 🛠️ Tech Stack
- Java
- Java Swing
- OOP
